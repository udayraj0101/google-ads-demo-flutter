# google-ads-demo-flutter

A Flutter project demonstrating Google Ads integration.

## Getting Started

This project showcases how to integrate and implement Google Ads in a Flutter application.

## Features

- Google Ads integration
- Flutter mobile development

## Setup

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app
